Recent Products Block for WooCommerce is a block plugin by which you can showcase your WooCommerce Recently added product in block Widgets, or anywhere in the block editor area. 

Discover an innovative way to exhibit your latest WooCommerce products effortlessly with the Recent Products Block for WooCommerce. Recent Products Block seamlessly integrates with the WordPress Gutenberg editor, empowering you to showcase your most recently added products with utmost flexibility.

Revolutionize your WooCommerce product display strategy with Recent Products Block for WooCommerce – where customization meets simplicity. With this plugin transform the way your customers explore your latest offerings!


**[Demos](https://bblockswp.com/demo/woocommerce-recent-products/)**


= Key Features – Free Version =
- **Universal Shortcodes**: Compatible with all page builders.
- **Query**: Display the product as your query. You can show products by the stock filter, category filter etc.
- **Layout**: Set different columns in different devices for responsiveness. Also, custom column and row gap.
- **Element Show/Hide**: Show or hide product elements, such as product image, title, rating, price, and add to cart button.

= Key Features – Pro Version =
- **Free + Pro Features**: The Pro version includes everything that’s available in the free version.
- **Universal Shortcodes**: Compatible with all page builders.
- **Templates**: Includes 2+ ready-to-use templates.
- **Element Visibility Control**: Show or hide any product element as needed.
- **Price & Discount Control**: Manage product prices and discount tags easily.
- **Color & Typography Settings**: Customize colors and typography for all elements.
- **Product Card Styling**: Control background, padding, and shadow options.
- **Image Customization**: Adjust image width, height, and object-fit settings.
- **Section Styling**: Customize section padding, margin, and background.
- **Review Count Display**: Show review counts with styling options.
- **Average Rating Display**: Control rating colors and typography.
- **Responsive Design**: Fully responsive and mobile-friendly layout.
- **Beginner-Friendly**: Easy to use and simple to customize.

= How to use =
- First, install the WooCommerce Recent Products plugin
- Add the WooCommerce Recent Products block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **WooCommerce Recent Products**
3. Click on the **WooCommerce Recent Products** to add the block

= Download & Upload: =
1. Download the **WooCommerce Recent Products** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`recent-products-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **WooCommerce Recent Products** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
