== Description ==

Recent Products Block for WooCommerce is a Gutenberg block plugin that allows you to showcase recently added WooCommerce products anywhere in your website using the block editor.

The plugin integrates seamlessly with the WordPress Gutenberg editor and provides flexible customization options to create attractive product showcases for your customers.




= Key Features =

* Shortcode support.
* Product query options.
* Filter products by category, stock status, and more.
* Responsive column settings for desktop, tablet, and mobile.
* Custom column and row gap controls.
* Show or hide product elements such as image, title, rating, price, and add to cart button.



= How to use =

1. Install and activate WooCommerce.
2. Install and activate the Recent Products Block plugin.
3. Open the Gutenberg editor.
4. Add the “WooCommerce Recent Products” block from the Widgets category.
5. Customize the block settings from the right sidebar.
6. Publish and enjoy.

== Installation ==

= From Gutenberg Editor =

1. Go to the WordPress Block Editor.
2. Search for “WooCommerce Recent Products”.
3. Click on the block to add it.

= Download & Upload =

1. Download the plugin ZIP file.
2. Go to Plugins > Add New in your WordPress admin dashboard.
3. Click “Upload Plugin”.
4. Upload the `recent-products-block.zip` file.
5. Click “Install Now”.
6. Activate the plugin.

= Manual Installation =

1. Download and extract the plugin files.
2. Upload the plugin folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the Plugins menu in WordPress.
